# Financial Data Automation Library 🐍

A collection of "Plug-and-Play" scripts designed to accelerate the digital transformation of finance departments.

### 🛠 Tools Included
- **Excel-to-SQL Pipeline:** Automated migration of legacy spreadsheets into relational databases.
- **PDF Scraper:** Extracting structured data from vendor invoices and financial statements.
- **Linux Workflow Tools:** Custom Bash aliases and shell scripts for managing data environments on Ubuntu.

### 🐧 Linux Advantage
Optimized for the **Ubuntu** terminal, ensuring high-speed processing and easy integration into server-side cron jobs.
